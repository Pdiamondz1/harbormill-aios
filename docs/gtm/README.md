# Harbormill GTM — 30-Day Playbook Assets

The runnable front-half of the [[Harbormill Ladder]]: how prospects are **found → converted →
onboarded → delivered → kept on retainer.** These assets operationalize the strategy in
`.claude/plans/` (30-Day Go-To-Market Playbook).

**Goal this push:** 2 signed Focused Projects → Retainers, in 30 days, solo, ~20–35 hrs/wk.
**Wedge:** AIOS deck (retainer engine) + AR/invoice follow-up (default first build), sold via the Loop Audit.

> **Go to market with confidence → [The Field Guide](field-guide/00-index.md).** The founder's
> entry point that sits on top of these assets: how to *know* the product, *pitch & demo* it, *run*
> the discovery call, and *close → onboard → operate*. Start at
> [`field-guide/00-index.md`](field-guide/00-index.md).

## The funnel (work backwards from 2 projects)

```
~50 warm contacts → ~12 intros → ~4–5 paid Loop Audits → 2 projects → 2 retainers
   (25%)              (35%)          (50%)
```

## Weekly cadence

- **Wk 1 — Pipeline:** Build Warm-50, tier it, outreach 10–15/day. *Target: 6–8 intros booked.*
- **Wk 2 — Convert #1:** Run intros, sell 2–3 audits, **close Project #1 + deposit.** Onboard #1.
- **Wk 3 — Deliver #1 / Convert #2:** Build #1, **close Project #2 + deposit.** Onboard #2.
- **Wk 4 — Complete #1 / start #2:** Ship #1, run completion ritual, **pitch retainer #1.** Start #2.

If intros lag in Wk 1, the bottleneck is list size / outreach volume — widen the list and lean on
referral asks *before* touching anything downstream.
