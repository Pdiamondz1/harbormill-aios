# Productized Retainer Tiers

Two fixed tiers so the retainer is repeatable, not negotiated from scratch each time. **Pitch at
delivery, when the project's ROI is fresh on the screen** (the deck's "Value Delivered" surface).
This is the scalable backend of the [[Harbormill Ladder]].

> Within the published band ($3,000–$10,000/mo). Defaults below; adjust to client scale.

---

## Tier 1 — **Operate** · $3,000/mo
*Keep what we built running and proving its worth.*
- AIOS deck operated + monitored; weekly AI brief delivered.
- The shipped automation (e.g. AR follow-up) kept healthy + watched.
- **Monthly ROI review** — value created vs. fee, on the deck.
- Email/async support.

## Tier 2 — **Operate + Build** · $5,000/mo
*Everything in Operate, plus the system compounds every month.*
- **One new automation "loop" built per month**, drawn from the client's ranked Loop Audit.
- Re-audit quarterly to refresh the candidate list.
- Priority support + a monthly working session.
- Expanding deck (new metrics, briefs, Aria tools) as loops ship.

*(Optional Tier 3 — Embedded · $8–10k/mo: multiple builds/mo, deeper integration, for clients
who've proven the model and want to move fast. Introduce only once Tier 2 is maxed.)*

---

## The pitch (at delivery)
> "The build's recovering ~$[X]/mo. The deck now watches that automatically and writes your weekly
> brief. For **$[3–5k]/mo** I keep it running, review the ROI with you each month, and we build the
> **next** loop from your audit — so the system keeps compounding instead of going stale. Most
> clients start on **Operate + Build** so something new ships every month."

Anchor on the proven ROI number: if the build returns $X/mo, a $3–5k/mo fee that protects *and grows*
it is an easy yes.

## Why this is scalable (the answer to "scalable retainer")
1. **Same flagship every client** — the AIOS deck template (deploy in hours, not weeks).
2. **Same method every client** — the Loop Audit / Four-Condition Test.
3. **Same monthly motion** — review ROI → pick next loop → build → deck reports it.
4. **Engine improvements flow upstream** — client #3 is cheaper to serve than client #1.
5. **Niche after revenue** — double down on the segment that converted; marketing compounds, CAC drops.

## Churn defense
The deck's **monthly value surface re-proves ROI every cycle.** As long as value > fee is visible,
the retainer renews itself. If a month's value dips, that's your signal to ship a higher-impact loop —
not wait for a cancellation.

## Sweven layer — agency niche + founding-client lever (added 2026-06-24)

The "niche after revenue" principle above now has a named target: the **beachhead agency sub-type**
from the [Sweven audit](sweven-audit.md). Agencies are higher-budget, so they typically land on
**Operate + Build** ($5,000) or **Embedded** ($8–10k) — *within the published band; no new tiers.*

**Founding-client lever (optional):** to seed the first 2–3 case studies the outreach currently lacks,
offer those early clients a reduced first-few-months retainer (or an extra credited audit) in exchange
for testimonial + case-study + referral rights. Keep it inside the $3–10k band and time-box it — it
buys proof, then ramps to standard pricing.

## Health metrics to track per retainer
- Value delivered vs. fee (target: ≥3× the monthly fee).
- Loops shipped per month (Tier 2: ≥1).
- Months retained; expansion (tier upgrades).
