# The Harbormill Field Guide

> **Read this before you go to market.** One documented operating system to understand the product,
> pitch and demo it, run a discovery call, and close — with confidence.

**Status:** complete

This guide *extends* the existing GTM system in [`../`](../README.md) (the ladder, scripts, and
templates in `docs/gtm/`). It adds the founder-facing layer on top of that funnel: how to *know*,
*say*, *run*, and *close* the product. Where this guide and `docs/gtm/` ever disagree on pricing or
the sales ladder, **`docs/gtm/` wins.**

## The deal lifecycle

The guide follows a single deal from cold to retainer. Read the four parts in order:

```
Know the Product  →  Pitch & Demo  →  Run the Call  →  Close / Onboard / Operate
   (Part 1)            (Part 2)         (Part 3)              (Part 4)
```

## Call tomorrow? Read these three

Short on time before a call, read only these — they cover the thesis, the product, and the live
moves:

1. [product-cheat-sheet.md](product-cheat-sheet.md) — the 10-second glance card (thesis, ladder, the
   four Loop conditions, the six surfaces).
2. [Part 1 — Know the Product](01-know-the-product.md) — so you can speak to every surface cold.
3. [Part 3 — Run the Call](03-run-the-call.md) — discovery questions, the prize-number worksheet, and
   objection handling for live use.

## Export to PDF (for your phone)

The two starred docs — the [cheat-sheet](product-cheat-sheet.md) and the [one-pager](one-pager.md) —
are meant to be carried as PDFs. Render either with any markdown→PDF tool (e.g. `pandoc product-cheat-sheet.md -o cheat-sheet.pdf`,
or just open it in a browser and **print → save as PDF**) and keep it on your phone for pre-call review.
