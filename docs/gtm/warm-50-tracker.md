# Warm-50 Pipeline Tracker

Your Week-1 deliverable. The funnel needs **~50 warm contacts reached** to land 2 projects.
Build this list first — everything downstream depends on it. (Copy into a Sheet or Airtable for
live status tracking; this markdown is the schema + seed.)

## How to build the list (pull from every source)

- **Ex-enterprise colleagues** — Nike, Assurant, IAA, LEGACY Supply Chain. Many have moved to
  SMBs or know owners. They won't all buy, but they're prime *referrers*.
- **Independent-consulting contacts** (2015→now) — closest to your SMB ICP; warmest path to a yes.
- **SUNY Delhi alumni**, local business owners, personal/community network.
- **Anyone who already knows you run automation work.**

## Tiering

- **A — likely buyer:** owns/runs an SMB with obvious repeating pain (AR, leads, reporting). Reach first.
- **B — likely referrer:** trusted, well-connected, not the buyer. Ask for 2 intros.
- **C — long shot:** worth one touch, low priority.

## Tracker schema

| Name | How I know them | Business / role | Likely pain (AR / leads / reporting) | Tier (A/B/C) | Warmth (1–3) | Ask (intro / referral) | Status | Next action | Date |
|---|---|---|---|---|---|---|---|---|---|
| _e.g._ Jane Doe | Ex-Assurant | Owns an HVAC co | AR — slow invoice pay | A | 3 | Intro call | Not contacted | Send DM | — |
| | | | | | | | | | |

**Status values:** `Not contacted → Contacted → Replied → Intro booked → Audit sold → Project signed → Retainer`

## Daily discipline

- ~10–15 personalized sends/day, **A-tier first.**
- Log every reply same day; book the call before the thread cools.
- **Every non-buyer gets the referral ask** — that's how a list of 30 becomes a funnel of 50.
